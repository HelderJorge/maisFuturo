Cobol - Leitura e Processamento de Dados recebidos de .txt

 IDENTIFICATION DIVISION.
PROGRAM-ID. FileTurma as "NotasCobolv01.cbl".
environment division.
input-output section.
file-control.
select FT assign to "Turma.txt"
organization is line sequential.

DATA DIVISION.


FILE SECTION.
fd FT.
01 Turma.
05 Numero Pic 9(2).
05 Nome Pic X(35).
05 Telefone Pic 9(9).
05 Mod14 Pic 9(2).
05 Nota14 Pic 9(2).
05 Mod15 Pic 9(2).
05 Nota15 Pic 9(2).

WORKING-STORAGE SECTION.
01 condicao pic 9 value zero.
01 linha-ecran pic 99 value zeros.
01 semvalor pic x.
01 avgi pic 99V99.
01 nregisto pic 99 value zeros.
01 total14 pic 9999 value zeros.
01 avg14 pic 99V99 value zeros.
01 min14 pic 99 value zeros.
01 max14 pic 99 value zeros.
01 total15 pic 9999 value zeros.
01 avg15 pic 99V99 value zeros.
01 min15 pic 99 value zeros.
01 max15 pic 99 value zeros.

PROCEDURE DIVISION.
Inicio.
open input FT.
Perform lerficheiro until condicao=1.
accept semvalor.
display space.
perform mostrarTotalizadores.
close FT.
Display "Obrigado!".

fim-programa.
stop run.

lerficheiro.
read FT at end move 1 to condicao.
if condicao = 0
perform registos.

registos.
add 1 to linha-ecran.
add 1 to nregisto.
display "Nº: "Numero.
display "Nome: " Nome.
Compute avgi=(Nota14+Nota15)/2
Display "Média: "avgi
If Nota14 > 9 and Nota15 > 9
Display "Módulos Não validados:---"
End-if
If (Nota14 > 9 and Nota15 <10) or (Nota14<10 and nota15>9)
Display "Módulos não validados: 1"
End-if
If Nota14<10 and Nota15<10
Display "Módulos não validados: 2"
End-if
Compute total14 = total14 + Nota14
Compute total15 = total15 + nota15
If nregisto = 1 then
Move Nota14 to min14
Move Nota14 to max14
Move Nota15 to min15
Move Nota15 to max15
Else
If Nota14 <= min14 Move Nota14 to min14 END-IF
If Nota14 >= max14 Move Nota14 to max14 End-if
If Nota15 <= min15 Move Nota15 to min15 End-if
If Nota15 >= max15 Move Nota15 to max15 End-if
End-if
++ Display space
if linha-ecran = 10
display "Carrega na tecla Enter."
ACCEPT semvalor
move 0 to linha-ecran
End-if.

mostrarTotalizadores.
compute avg14 = total14/nregisto
display "Média14: " avg14
display "Max14: " max14
display "min14: " min14
compute avg15 = total15/nregisto
display "Média15: " avg15
display "Max15: " Max15
display "min15: " min15.

END PROGRAM FileTurma.
